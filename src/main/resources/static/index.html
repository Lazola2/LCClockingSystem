Clocking System

API endpoints for Life Choices clocking system

Method Path Description
/ get Root endpoint
/users/all get get all users
/users/user/:id get get a user specified by id
/users/add post register a user
/users/update put update a user
/users/login post loggin in
/users/user/:id delete delete a user specified by id
/users/all delete delete all users
/clocking/add post add a clocking record
/clocking/all/user/:id get get all clocking record for a specified user id
/clocking/update put update a clocking record
